To alert Host when their visitor arrives, QeyPlus needs to "know" two pieces of information: First, which Host is the visitor meeting? And second, what is that Host's contact information?

When the visitor signs in, the visitor will type the Host's name in the host field. Then, QeyPlus will locate the employee's contact information in the directory and send the host notification.

Benefits of host notifications

Save the time it takes to call or track down employees when their visitors arrive. Photos appear in host notifications, so it's easy to know which visitor to greet in the lobby. The host can choose their preferred notification methods.

Notification methods

As the Envoy administrator, you'll choose the default host notification methods for your location, but employees can also set their own host notification preferences.

QeyPlus Provide Following Notification Methods

  • QeyPlus Push Notifications
  • SMS Notification
  • Whatsapp Notifications

Go to Settings > Visitor Type > <Select Visitor Type>.

User can Enable / Disable "Host SMS Notification" or "Host Whatsapp Notification"

It's easy to customize the text that appears in host notifications. You can add custom variables to automatically populate the host's name, visitor's name, company, and location names.

You can customize the notification text that assistants and fallback contacts receive, too. This allows you to provide more detailed greeting instructions for cases where the host may not be the person greeting the visitor.
